A animação acima representa o Quick Sort. Nesse exemplo o último elemento é escolhido como pivô


Dentro do código que realiza a partição dos valores que estão dentro de um vetor há uma comparação, para que um elemento seja alterado de posição ele irá comparar o atual elemento dado pelo índice "i" e comparar com o de índice "j", em que o valor de j é o dobro o valor de i sendo que quando i = 0 -> j = 1.

Com isso todos os elementos estarão ordenados fora o elemento pivô, que por sua vez irá ser trocado com o elemento i + 1, assim desordenado novamente os três últimos elementos, que serão particionados e ordenados e com isso sendo inseridos no final do vetor com todos os elementos finalmente ordenados.
